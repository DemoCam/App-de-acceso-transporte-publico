Guía de uso

Consultando rutas

Para consultar las rutas disponibles:

  1. Accede a la sección "Rutas" desde el menú principal.
  2. Utiliza el buscador para filtrar por número o nombre de ruta.
  3. Haz clic en "Ver detalles" para obtener información detallada de una ruta específica.

Consultando paradas

Para consultar las paradas disponibles:

  1. Accede a la sección "Paradas" desde el menú principal.
  2. Utiliza el mapa interactivo para visualizar la ubicación de las paradas.
  3. Utiliza los filtros para encontrar paradas con características específicas de accesibilidad.

Creando una cuenta

Para crear una cuenta y acceder a funciones adicionales:

  1. Haz clic en "Registrarse" en la barra de navegación.
  2. Completa el formulario con tus datos.
  3. Inicia sesión con tus credenciales.

Opciones de accesibilidad

Alto contraste

Para activar el modo de alto contraste, haz clic en el botón ubicado en la barra de navegación.

Tamaño de texto

Para aumentar el tamaño del texto, haz clic en el botón ubicado en la barra de navegación.

Navegación por teclado

Esta plataforma es completamente navegable utilizando el teclado. Utiliza la tecla Tab para desplazarte entre elementos y Enter para activarlos.

Lector de pantalla

Este sitio es compatible con lectores de pantalla como JAWS, NVDA y VoiceOver.

Asistente de voz

El asistente de voz te permite interactuar con la plataforma mediante comandos hablados. Para activarlo:

Comandos disponibles

  • "Ir a inicio" - Navega a la página principal
  • "Mostrar rutas" - Navega a la página de rutas
  • "Mostrar paradas" - Navega a la página de paradas
  • "Buscar [término]" - Realiza una búsqueda
  • "Ayuda" - Muestra esta página de ayuda
El asistente de voz requiere permisos de micrófono en tu navegador.

Preguntas frecuentes

No, la consulta de rutas y paradas está disponible para todos los usuarios. El registro te permite acceder a funcionalidades adicionales como guardar rutas favoritas.

Las paradas con características de accesibilidad están marcadas con iconos específicos en el listado y el mapa. Puedes filtrar por paradas con rampas o semáforos sonoros utilizando los filtros disponibles.

Actualmente mostramos los horarios programados y la frecuencia de cada ruta. Estamos trabajando para implementar información en tiempo real en futuras actualizaciones.
